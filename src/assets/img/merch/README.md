# Merch product photos

Drop product photos here, then set each item's `photo` field in
`src/_data/merch.json` to the matching path. Until `photo` is set, the page
shows a "Photo coming soon" placeholder.

Expected files (square images work best, e.g. 800×800):

| Product | File | merch.json `photo` value |
|---|---|---|
| Work Shirt | `work-shirt.jpg` | `/assets/img/merch/work-shirt.jpg` |
| Black T-Shirt | `black-tee.jpg` | `/assets/img/merch/black-tee.jpg` |
| Pullover Hoodie | `hoodie.jpg` | `/assets/img/merch/hoodie.jpg` |
| Cap | `cap.jpg` | `/assets/img/merch/cap.jpg` |
